# C++ Practice Repository

This repository contains my **C++ practice programs** created while learning programming fundamentals, problem solving, and algorithms.

The goal of this repository is to improve my understanding of **C++ programming concepts** and build a strong foundation in computer science.

---

## Topics Covered

The programs in this repository include practice on:

- Basic C++ Syntax
- Variables and Data Types
- Input and Output
- Conditional Statements (if / else)
- Loops (for, while, do-while)
- Functions
- Arrays
- Pointers
- Basic Algorithms
- Mathematical Programs
